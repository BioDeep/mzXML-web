Instructions

  • 1. First of all, select a raw file on left panel
  • 2. Then TIC plot will be loaded for the file that you've selected.
  • 3. TIC plot is show for your entire Raw File by default.
  • 4. Select precursor m/z for view TIC plot of the ions under specific m/z
  • 5. A red dot on TIC plot is a [mz,rt] ion, click on it to view corresponding ms2 spectrum.
  • 6. Set m/z min and max to draw selective ions TIC plot.
